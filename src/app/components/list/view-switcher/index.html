The list view switcher module displays toolbar options that enable users to switch between different views of the same data.

Use when

Use list view switchers when users need to switch between different views of the same data.
Do use list view switchers when more than one view of data is available to users.

Don't use when

Don't use list view switchers when only one view of data is available.
Don't use list view switchers when users cannot switch between different views.
 Active view button Inactive view button

Available views

You must specify the views that are available to users.

Default view

You must specify the default view to display when the page loads.

Custom views

You can add custom views if the available SKY UX list types don't work for your scenario.

Switch view

When users switch views, the view changes while filters remain applied.

Responsiveness

On small viewports, the view switcher collapses to a dropdown to conserve space.
