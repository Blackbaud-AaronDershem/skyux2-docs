Default country

By default, the country selector will be set to the United States. You can specify a different default value or set the default dynamically, for example by using location detection.

Country selection

If the user is entering a phone number from a country other than that currently selected, they must choose the correct country to ensure the phone number passes validation.

Phone format validation

The phone input is validated against the format expected for the selected country. You have to specify the error message; in most cases the error message should say "Enter a valid phone number."

Page titles

If tabs are the main navigation on a page, display the tab header in the page title. Use the following format: <Page title> — <Company, product, or site identifier>. For example: Portfolio — Work center — Blackbaud. In addition, follow these guidelines for page titles:

  • Keep titles short.
  • Make titles unique.
  • Provide concise descriptions of each page's contents or purpose. Titles should stand on their own and make sense out of context.
  • Start with the most meaningful, differentiating terms for users to scan in browser tabs, window lists, and browser bookmarks.

For information about how to use the Angular Title service and the SKY UX title configuration property to specify the title, see the configuration reference.
